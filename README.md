# Grocery Billing System Case Study

## Objective
To develop a Python program that calculates grocery bills.

## Problem Statement
The system accepts prices of 5 items and calculates total cost.

## Algorithm
1. Initialize total = 0
2. Accept price of 5 items
3. Add each price to total
4. Display total

## Code Explanation
The program uses loop, input function, and arithmetic operations.

## Output
Displays total grocery bill.
